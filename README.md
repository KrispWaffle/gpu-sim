# Syntax

It goes 
- Operation 
    - Destination

    - Value (if required)

Once you have your program add a halt to let GPU know its the end

# Instructions
- ADD
- SUB
- MUL
- NEG 
- LD (load)
- ST (store)
- MOV 
- HALT (end of program)
